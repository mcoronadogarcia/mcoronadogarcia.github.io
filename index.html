Introduction to Geographic Visualization Final Project

Project Description

The overall goal of this project was to determine a creative method to display Opioid Overdose data in the United States over time. Originally, I planned to create a dorling cartogram to display this data; however, after class lecture, I realized that I would need a second variable to explore along with the opioid overdose death data and time data. Unfortunately, I decided that there were no additional variables available to me that would help me create a meaningful plot. Instead, I focused on using choropleth maps in the way we used them for assignments. With these maps and using my data, I was able to find interesting relationships. Additionally, I attempted to make a plot that allowed you to plot trajectories based on a states opioid death rates as time continued. Unfortunately, Everything was working beside the line trajectories and navigation bar layout. More work needs to be done on the last project, however, it does still display the crude death rates of each state to and these change as you hover your mouse over the x axis of the plot.
